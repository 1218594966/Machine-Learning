道场指引：如何开启修仙旅程

按照下方步骤配置环境、启动站点，即可进入沉浸式修仙教程。所有命令均以 bash 形式展示，Windows 用户可在 PowerShell 或 Git Bash 中执行。

① 获取法器

git clone https://github.com/<your-name>/Machine-Learning.git
cd Machine-Learning
python -m venv .venv
source .venv/bin/activate  # Windows 使用 .venv\Scripts\activate
pip install -r requirements.txt

若下载缓慢，可将 pip 镜像切换到清华或阿里云源。

② 启动修仙道场

uvicorn app.main:app --host 0.0.0.0 --port 8000

浏览器访问 http://127.0.0.1:8000 查看教程；若需热更新可添加 --reload。

③ 修炼方式

  • 阅读 README 作为随身心法，了解全局路线。
  • 在网站中按章节修炼，完成练气题库与阶段试炼。
  • 使用 Git 记录修行历程，并撰写每周复盘。

  • 完成 VS Code、Git、虚拟环境的配置。

  • 版本控制：git init、git add、git commit、远程仓库推送。

  2. 使用 pip 安装 numpy 与 pandas，打印版本验证。
  3. 在 VS Code 中运行 hello_cultivator.py，输出自我介绍。
  4. 初始化 Git 仓库并推送至 GitHub，记录环境搭建笔记。
  5. 撰写《凡尘初醒》心得，列出遇到的问题与解决方案。

练气题库

常见疑问与解析
  • 创建文件夹并进入：在终端执行 mkdir dao-field 后 cd dao-field，若报错可先运行 pwd 查看当前路径。
  • 升级 pip：激活虚拟环境后运行 python -m pip install --upgrade pip，如遇 SSL 错误，可设置镜像。
  • VS Code 解释器：使用 Ctrl+Shift+P → “Python: Select Interpreter” 选择 .venv。
  • Git 提交：git commit -m "feat: first step" 前确保配置了 user.name 与 user.email。
  • 推送远程：添加远程后执行 git push -u origin main，如需 Token，可在 GitHub Developer Settings 生成。

灵根觉醒 · 基础数据类型与变量
  1. 题目：声明 spirit_name、age、is_novice 并打印类型。
    点拨： 使用 type()。
    解析： 了解字符串、整数与布尔的基本差异。
  2. 题目：将字符串 "42" 与整数 10 相加。
    点拨： 先做类型转换。
    解析： 使用 int("42") + 10，避免 TypeError。
  3. 题目：验证 0.1 + 0.2 == 0.3 是否为真。
    点拨： 尝试 math.isclose。
    解析： 解释浮点误差并使用 math.isclose(0.1 + 0.2, 0.3)。
  4. 题目：编写脚本交换两个变量的值。
    点拨： 多重赋值。
    解析： a, b = b, a 展示 Python 的优雅语法。
  5. 题目：比较 None 与空字符串在布尔上下文中的差异。
    点拨： 使用 bool()。
    解析： None 表示无值，空字符串虽为 False 但有类型与长度，掌握真假值判断。
灵气流转 · 运算符与表达式

  2. 题目：修为每日增长 10%，7 天后是多少？点拨： 幂运算。解析： 100 * (1.1 ** 7)。
  3. 题目：判断修士等级是否在 10-50 且非 30。点拨： 链式比较。解析： 10 <= level <= 50 and level != 30。

阵法初成 · 控制流与推导式
  1. 题目：打印 1-100 中能被 7 整除的数字。
    点拨： 使用 for + range。
    解析： for num in range(1, 101): if num % 7 == 0: print(num)，熟悉基础循环。
  2. 题目：设计 5 次机会的猜数字小游戏。
    点拨： 结合 random.randint 与 while 的 else。
    解析： 借助计数器与 break 控制回合，失败时在 else 中提示答案。
  3. 题目：用列表推导式生成奇数平方列表。
    点拨： 条件表达式过滤偶数。
    解析： [x**2 for x in range(1, 21) if x % 2]，演示推导式精简写法。
  4. 题目：拍平嵌套列表成为一维序列。
    点拨： 双层推导式或 itertools.chain。
    解析： [item for sub in nested for item in sub]，理解遍历顺序。
  5. 题目：借助 enumerate 和 continue 跳过空白输入。
    点拨： 对列表遍历同时输出序号。
    解析： 当元素为空字符串时 continue，否则打印序号与姓名。
丹田扩展 · 序列与集合

    解析： 展示列表方法的返回值差异，并解释就地修改与返回新列表的区别。
  2. 题目：统计灵药库存的最高、最低与平均值。
    点拨： 结合 max、min、sum/len。

  3. 题目：将元素元组转换为列表后追加新元素。
    点拨： 使用内置 list()。
    解析： elements = list(elements_tuple)，强调元组不可变但可通过新列表扩展。
  4. 题目：比较两名弟子的法术集合，输出交集、并集与差集。
    点拨： 使用集合运算。

灵术传承 · 函数与参数

气海调息 · 匿名函数与迭代器
  1. 题目：利用 map + lambda 计算灵药含税价格。
    点拨： 列表传入 map。

  2. 题目：使用 filter 挑选修为超过 500 的弟子。
    点拨： 传入元组列表。
    解析： list(filter(lambda x: x[1] > 500, disciples))，演示匿名函数。
  3. 题目：使用 sorted 的 key 参数实现“修为降序、年龄升序”排序。
    点拨： lambda 返回元组。

  4. 题目：定义生成器按境界产出任务。
    点拨： 使用 yield from。
