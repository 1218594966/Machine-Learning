练习 1：判断闰年

任务说明：编写程序判断用户输入的年份是否为闰年。

知识回顾：闰年规则——被 4 整除但不被 100 整除，或能被 400 整除。

year = int(input("请输入年份:"))
if (year % 4 == 0 and year % 100 != 0) or year % 400 == 0:
    print("闰年")
else:
    print("平年")

常见错误解析：不要忘记使用 int() 将字符串转为整数，否则无法求余。

练习 2：统计句子中的单词次数

任务说明：输入一句英文句子，统计每个单词出现的次数。

sentence = input("请输入一句话:")
words = sentence.lower().split()
count = {}
for word in words:
    count[word] = count.get(word, 0) + 1
print(count)

常见错误解析：记得使用 lower() 将单词统一为小写，避免大小写导致重复计数。

练习 3：面向对象封装学生成绩

任务说明：设计 Student 类，支持添加成绩并打印“姓名-平均分”。

class Student:
    def __init__(self, name):
        self.name = name
        self.scores = []

    def add_score(self, score):
        self.scores.append(score)

    def average(self):
        return sum(self.scores) / len(self.scores)

    def __str__(self):
        return f"{self.name}-{self.average():.1f}"

常见错误解析：注意在类中使用 self 引用实例属性，且平均值需防止除以零，可先判断是否有成绩。

练习 4：使用 Pipeline 串联预处理与建模

任务说明：使用 ColumnTransformer + Pipeline 构建标准化与随机森林回归的流水线。

from sklearn.compose import ColumnTransformer
from sklearn.pipeline import Pipeline
from sklearn.preprocessing import OneHotEncoder, StandardScaler
from sklearn.ensemble import RandomForestRegressor

numeric_features = ['age', 'fare']
categorical_features = ['sex', 'embarked']

preprocess = ColumnTransformer([
    ('num', StandardScaler(), numeric_features),
    ('cat', OneHotEncoder(handle_unknown='ignore'), categorical_features)
])

pipe = Pipeline([
    ('preprocess', preprocess),
    ('model', RandomForestRegressor(random_state=42))
])
pipe.fit(train_df[numeric_features + categorical_features], train_df['survived'])

常见错误解析：使用 handle_unknown='ignore' 处理测试集中未出现的类别，避免预测时报错。

练习 1：创建张量并计算求和

任务说明：使用 PyTorch 创建一个 3×3 的随机张量并计算求和。

import torch
x = torch.rand((3, 3))
print(x)
print(x.sum())

常见错误解析：如需复现结果可设置 torch.manual_seed(42) 固定随机数。

练习 2：构建两层全连接网络

任务说明：使用 nn.Sequential 构建两层网络并完成前向传播。

import torch.nn as nn
model = nn.Sequential(
    nn.Linear(4, 8),
    nn.ReLU(),
    nn.Linear(8, 1)
)
sample = torch.randn(2, 4)
print(model(sample))

常见错误解析：线性层输入尺寸需与样本特征维度一致，报错时检查 Linear(in_features, out_features) 的参数。

练习 3：编写训练循环并记录损失

任务说明：实现基础训练循环并打印每轮损失。

for epoch in range(5):
    optimizer.zero_grad()
    outputs = model(inputs)
    loss = criterion(outputs, labels)
    loss.backward()
    optimizer.step()
    print(f"Epoch {epoch+1}: loss={loss.item():.4f}")

常见错误解析：确保调用 optimizer.zero_grad() 清空梯度，避免梯度累积。

练习 4：评估准确率并保存模型

任务说明：计算模型在验证集上的准确率，并保存权重文件。

with torch.no_grad():
    preds = model(val_inputs).argmax(dim=1)
    acc = (preds == val_labels).float().mean().item()
    print(f"验证准确率：{acc:.3f}")

torch.save(model.state_dict(), 'model.pt')

常见错误解析：推理阶段需加 torch.no_grad() 关闭梯度；保存模型权重使用 state_dict() 更灵活。
