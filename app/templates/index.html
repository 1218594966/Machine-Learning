道场指引：如何开启修仙之旅

按照以下三步就能启动修仙网站与随身心法。命令以 bash 写成，Windows 道友可使用 PowerShell 或 Git Bash。

① 领取法器

git clone https://github.com/<your-name>/Machine-Learning.git
cd Machine-Learning
python -m venv .venv
source .venv/bin/activate  # Windows 使用 .venv\Scripts\activate
pip install -r requirements.txt

若下载缓慢，可设置 pip 镜像：pip config set global.index-url https://pypi.tuna.tsinghua.edu.cn/simple。

② 启动道场

uvicorn app.main:app --host 0.0.0.0 --port 8000 --reload

浏览器访问 http://127.0.0.1:8000，左侧目录会展示五境导航。

③ 修炼方式

  • 将 README 当作随身心法，对照网站完成任务。
  • 每个知识点至少完成 5 道练气题，再阅读解析巩固。
  • 使用 Git 记录修炼历程，形成周志与阶段复盘。

答疑密信 · 常见问题

Q1：完全零基础可以学吗？

可以。按照练气境的顺序逐步完成。遇到陌生术语，可查阅 README 与附录资源。

Q2：题目太多做不完怎么办？

建议全部练习；若时间有限，至少完成每个知识点前三题并记录解析，待有余力再补全。

Q3：需要哪些数学基础？

至少掌握高中函数、微积分入门与概率统计；在元婴境开始前逐步补足相关知识。

Q4：没有 GPU 如何修炼化神境？

可使用 Kaggle Notebook、Google Colab 或租用云 GPU；亦可在 CPU 上练习小数据集与推理流程。

Q5：完成五境后做什么？

深入研究 NLP、CV、强化学习等专精方向，参与开源项目或比赛，持续打磨内功。
