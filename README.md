## 道场指引：如何使用本仓库

### 1. 获取法器

```bash
# 克隆仓库
git clone https://github.com/<your-name>/Machine-Learning.git
cd Machine-Learning

# 创建并激活修行虚拟环境
python -m venv .venv
source .venv/bin/activate  # Windows 使用 .venv\Scripts\activate

# 装备所需灵材
pip install -r requirements.txt
```

### 2. 启动修仙道场（本地学习网站）

```bash
uvicorn app.main:app --host 0.0.0.0 --port 8000
```

- 启动后在浏览器访问 <http://127.0.0.1:8000>。
- 导航栏展示修行目录，可按阶段阅读教程、训练题与解析。
- 若需热更新，可使用 `--reload` 参数。

### 3. 道法使用方式

1. **阅读 README** 获取完整路线、题库与资源，作为随身心法。
2. **打开网站** 体验沉浸式修仙界面，按章节逐一修炼。
3. **完成试炼**：每个知识点至少亲手完成题目与项目，并用 Git 记录心得。
4. **阶段复盘**：每周写一篇复盘日志，记录体悟与突破。
